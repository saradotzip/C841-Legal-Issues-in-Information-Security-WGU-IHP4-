## Case Study Summary (TechFite)

TechFite is a NASDAQ-traded company whose Applications Division engaged in:

- Unauthorized access to external networks using Metasploit (ECPA, CFAA violations)
- Creation of dummy user accounts with escalated privileges (CFAA)
- Misappropriation of confidential client information from Orange Leaf Software LLC and Union City Electronic Ventures
- Shell company fraud through Bebop Software, FGH Research Group, and Dazzling Comet Software (SOX)
- Negligent internal security reporting by IT Security Analyst Nadia Johnson (conflict of interest)
- Physical surveillance ("dumpster diving") coordinated by BI Unit analyst Jack Hudson
